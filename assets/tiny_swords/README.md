# Tiny Swords Assets

Place the downloaded and extracted `Tiny Swords (Free Pack).zip` contents in this directory.

The current prototype is fully playable without the art pack by using small built-in pixel-style shapes. After the pack is available, replace the procedural bodies and map drawing with the matching unit, terrain, decoration, and UI sprites from this folder.

Source: https://pixelfrog-assets.itch.io/tiny-swords
